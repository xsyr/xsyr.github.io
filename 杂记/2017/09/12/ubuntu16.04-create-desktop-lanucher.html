Ubuntu 16.0创建启动图标

黄成林 12 Sep 2017

1. 安装 gnome-panel

$ sudo apt install gnome-panel

2. 创建启动图标

$ gnome-desktop-item-edit --create-new ~/Desktop

填写名称（启动器的名称），命令，点击ok可以创建启动图标。

3. 编辑启动图标在桌面的名称

编辑名称和描述。

4. 复制到 /usr/share/applications，使其可在 Dash(Win键)启动

$ cp xxx.desktop /usr/share/applications
